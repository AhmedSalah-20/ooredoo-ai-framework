# ⚙️ Framework de Fine-Tuning et d’Expérimentation pour LLMs

## 📌 Présentation du Projet
Conception d'une plateforme standardisée et modulaire permettant d'adapter efficacement des modèles de langage open-source à des cas d'usage spécifiques, tout en garantissant la reproductibilité et l'évaluation objective des performances.

## 🛠️ Stack Technique
* **LLM Core & FT:** Python, Hugging Face (Transformers, PEFT, TRL), Axolotl, PyTorch
* **Scaling & Optimization:** DeepSpeed, FSDP, Optuna
* **MLOps & Tracking:** Weights & Biases, MLflow
* **DevOps:** Docker, GitHub Actions
